## Dashboard Highlights

- KPI cards (Total Reviews, Auto-Triaged, Negative Reviews, Avg Rating, SLA)
- Sentiment trend chart
- Top issues panel with progress bars
- Priority review queue table with platform badges
- Star-based rating display in each review row

## Interactions

- Time range dropdown (`Last 7 days`, `Last 30 days`, `Last 90 days`)
- Platform dropdown (`All`, `Amazon`, `Flipkart`, `Shopify`, `Google`)
- Topic click filter (left panel)
- Combined filtering (time + platform + topic)
- Empty state row when no matching data
- `Create Executive Report` download button

## How to Use

1. Open `wireframe-dashboard-v2.html` in any browser.
2. Change dropdowns and click topics to filter data.
3. Click `Create Executive Report` to download the filtered summary.
